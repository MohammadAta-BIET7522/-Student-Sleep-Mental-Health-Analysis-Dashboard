## 📌 Project Overview

An interactive Power BI dashboard analyzing student sleep, mental
health, academic performance, screen time, anxiety, and burnout using
KPIs, DAX measures, charts, and slicers.

## 🎯 Project Objectives

-   Analyze sleep patterns.
-   Study burnout and anxiety.
-   Compare GPA by education level.
-   Understand screen time impact.
-   Build an interactive BI dashboard.

## 💼 Business Problem

Help educators identify lifestyle factors affecting student wellness and
academic performance.

## 📊 Dashboard Features

### KPIs

-   Total Students
-   Average Sleep Hours
-   Average GPA
-   Average Anxiety Score
-   Average Study Hours
-   Average Physical Activity
-   Average Screen Time
-   Burnout Percentage

### Visuals

-   Burnout Donut Chart
-   Sleep Hours by Gender
-   GPA by Education Level
-   Screen Time vs Anxiety Scatter Plot

### Filters

Gender, Sleep App Usage, Burnout Status, Education Level

## 🧮 DAX Measures

Total Students, Average Sleep Hours, Average GPA, Average Anxiety Score,
Average Study Hours, Average Physical Activity, Average Screen Time,
Burnout Percentage.

## 🛠️ Tools

Power BI, Power Query, DAX, Excel, Data Modeling.

## 📈 Key Insights

-   Average sleep ≈ 7.4 hours.
-   Burnout affects many students.
-   Higher screen time relates to higher anxiety.
-   GPA varies by education level.
